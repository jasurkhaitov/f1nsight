# 💰 F1nsight

**F1nsight** helps you **track, predict, and understand your daily spending** with **AI-powered insights**, **smart budgeting**, and **personalized financial recommendations**.

---

## 🚀 Features

* 📊 **Expense Tracking** – Monitor your daily, weekly, and monthly spending effortlessly.
* 🤖 **AI Insights** – Get automatic analysis and predictions of your future expenses.
* 💡 **Smart Recommendations** – Receive personalized tips to optimize your budget.
* 🎯 **Goal Tracking** – Set financial goals and measure progress in real-time.
* 🌓 **Modern UI** – Minimal, responsive, and user-friendly interface.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite) + TypeScript + Tailwind CSS
* **State Management:** Redux/Toolkit
* **Backend:** Java
* **AI & Analytics:** Google Gemini

---

## 📦 Installation

```bash
# Clone the repo
git clone https://github.com/jasurkhaitov/f1nsight.git

# Go to project folder
cd f1nsight

# Install dependencies
npm install

# Run development server
npm run dev
```

---

## 🧠 Usage

1. Create or import your account.
2. Add your expenses and categorize them.
3. Explore AI insights and personalized recommendations.
4. Track goals and visualize trends through interactive charts.
